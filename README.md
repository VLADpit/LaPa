# LaPa - Lightweight Handwritten & Printed OCR Application

Веб-приложение для распознавания печатного и рукописного текста (русский/английский языки) на изображениях с возможностью экспорта в текстовые файлы и сохранения истории в локальной базе данных

## Стек технологий
* **AI/CV:** Python, OpenCV, EasyOCR / PaddleOCR 
* **GUI:** Streamlit 
* **Database:** SQLite
* **DevOps:** Docker

## Функционал
- Загрузка изображений через удобный графический интерфейс
- Предварительная обработка изображений (улучшение контраста, бинаризация) для повышения точности OCR
- Распознавание смешанного текста (печатный + рукописный, RU/EN)
- Редактирование распознанного текста прямо в интерфейсе перед сохранением
- Сохранение результатов (дата, имя файла, распознанный текст) в локальную БД SQLite
- Скачивание результата в формате `.txt`
